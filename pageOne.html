The twit Library

In this guide we will be using one of the more popular libraries called twit. Now you may be asking what is a library and why are we using this for our bot. Both are good questions. A library is nothing more than a set list of premade commands that allows us (the user) to access Twitters API’s in a more efficient manner. Essentially, it’s premade code that simplifies our programs access to Twitters API’s and allows us to interact with them very easily.


twit is actively maintained and has been downloaded this month alone (at of the writing of this guide) close to 33,000 times.


Assuming you already have node installed, create a new folder to hold your bot. At this point the author of this guide recommends you also make a JSON package for your bot. Simply type npm init and follow the guide on the prompter. Once created, we will add the twit library.


Still within the recently create folder and in your command prompt, type npm install twit –save


That is it, the library has been downloaded for your bot, and has been added to your JSON package's dependencies.
